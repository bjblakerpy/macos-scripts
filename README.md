# MacOS Utility Scripts

A collection of automation and maintenance scripts designed to streamline workflows, manage installations, and keep MacOS systems running smoothly.

## üöÄ Getting Started

### Prerequisites
- Most scripts require **Homebrew** to be installed.
- Ensure you have the necessary permissions to run bash scripts.

### Installation
1. Clone this repository:
   ```bash
   git clone https://github.com/bjblakerpy/macos-scripts.git
   cd macos-scripts
   ```
2. Make scripts executable:
   ```bash
   chmod +x *.sh
   ```

## üõ† Included Scripts

| Script | Description |
| :--- | :--- |
| `brew_update.sh` | Updates Homebrew, upgrades all formulae/casks, and cleans up old versions. |
| `install_my_apps.sh` | Automates the installation of a curated list of essential MacOS applications. |

## üß¨ Extending the Collection
This repository is a living collection. To add new scripts:
1. Drop your `.sh` (or other script) file into the root directory.
2. Ensure the script is well-commented.
3. (Optional) Group specialized scripts into subdirectories if the collection grows significantly.
